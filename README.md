Starbuck
项目描述
针对此项目，建立一个模型来对星巴克的推送成功率进行预测。
      
Installation
Must runing with Python 3 with libraries of numpy, pandas, sqlalchemy, re, NLTK, pickle, Sklearn, plotly and flask libraries.
